### Projeto Agência Vision Comunicação

#### Página criada com base na seguinte [interface](https://xd.adobe.com/view/375e9250-3f21-4fdf-9a2c-528521861ca8-620d/specs/)

#### Tecnologias Utilizadas
- HTML: Utilizado para estruturar o conteúdo do site.
- CSS: Estilização dos elementos HTML para uma apresentação visual atraente.
- JavaScript: Adiciona interatividade e funcionalidades dinâmicas ao site.
- Bootstrap: Framework CSS utilizado para facilitar o desenvolvimento responsivo e o design de interface.
